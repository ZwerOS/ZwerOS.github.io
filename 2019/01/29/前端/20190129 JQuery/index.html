10.3 闭包原理

好处：

1、可以减少全局变量的对象，防止全局变量过去庞大，导致难以维护

2、防止可修改变量，因为内部的变量外部是无法访问的，并且也不可修改的。安全

3、读取函数内部的变量，另一个就是让这些变量的值始终保持在内存中。

11.4 总结

  1. 选择器的选择

    1. 单选框、文本框、按钮可以选择用 ID 选择器，也是最常用的方式
    2. 多选框有多个可以选择使用 属性 Attitude 选择器与表单属性对象选择器混用
    3. 节点的值可以使用parent() 、childrend() 、eq()方法

  2. each()方法是对节点集合的遍历，相比使用 for更加便捷

  3. 不可将 JQuery 对象与 JS相混

    注意在节点使用事件触发函数传入this 指针时，用 JQuery 操作需要将 JS 对象转为 JQuery对象用$(th)
